# transformeTexto
Extensão para formatar texto na versão 2.0

## Funcionalidade

- Textos Transformados para Letras Maiúsculas
- Textos Transformados para Letras Minúsculas
- Textos Transformados com a Primeira Letra Maiúscula após os pontos: ! ? . ;
- Limpar o conteúdo na entrada e na saída de dados
- Tema dark e light
